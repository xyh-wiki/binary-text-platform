# SEO / robots / sitemap 说明

- 子域 `binaryextract.xyh.wiki` 的 SEO 文件：
  - `frontend/binary-text-frontend/robots.txt`
  - `frontend/binary-text-frontend/sitemap.xml`
  - `frontend/binary-text-frontend/ads.txt`
  - `frontend/binary-text-frontend/nginx.conf`
- Docker 部署时，前端 Dockerfile 会将这些文件复制到 Nginx 静态目录。
